## A. Flow Overview

### Purpose
Users create, edit, publish, and manage content posts.

### Entry Points
- Dashboard → "New Post" button
- Posts page → "New Post" button
- Direct navigation to `/:orgId/posts/new`

### Steps
1. **Create Post**
   - Navigate to create page
   - Fill in post details
   - Add content (editor)
   - Set metadata
   - Save draft or publish

2. **Edit Post**
   - Navigate from posts list
   - Edit content
   - Update metadata
   - Save changes

3. **Publish Post**
   - From draft state
   - Review before publishing
   - Publish action

4. **Delete Post**
   - From posts list
   - Confirmation
   - Delete action
