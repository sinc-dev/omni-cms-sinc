## A. Flow Overview

### Purpose
Administrators invite users, assign roles, and manage team members.

### Entry Points
- Users page → "Invite User" button

### Steps

1. **Invite User**
   - Click "Invite User"
   - Enter email
   - Select role
   - Send invitation

2. **Manage User Roles**
   - View users list
   - Edit user role
   - Update permissions

3. **Remove User**
   - Remove from organization
   - Confirm action
