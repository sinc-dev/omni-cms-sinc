## A. Flow Overview

### Purpose
Users upload, organize, and manage media files (images, videos, documents).

### Entry Points
- Media page → "Upload" button
- Post editor → Media picker
- Drag-and-drop upload

### Steps

1. **Upload Media**
   - Click upload button or drag-and-drop
   - Select files
   - Upload progress
   - Success/error feedback

2. **Organize Media**
   - Browse media library
   - Filter by type, date, uploader
   - Search by filename

3. **Use Media in Content**
   - From media picker in editor
   - Select existing media
   - Insert into content

4. **Delete Media**
   - From media library
   - Confirmation
   - Delete action
